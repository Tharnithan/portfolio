## üñºÔ∏è Image Specifications

### 1. Profile Image (Hero Section)
- **Filename:** `profile.jpg`
- **Location:** `images/profile.jpg`
- **Recommended Size:** 600x600px (square)
- **Format:** JPG or PNG
- **Max File Size:** 150KB
- **Content:** Professional headshot or portrait
- **Tips:**
  - Use a clean, simple background
  - Good lighting on your face
  - Professional attire
  - Confident, friendly expression

### 2. About Image
- **Filename:** `about.jpg`
- **Location:** `images/about.jpg`
- **Recommended Size:** 500x700px (portrait orientation)
- **Format:** JPG or PNG
- **Max File Size:** 200KB
- **Content:** Full body or upper body shot, or working environment
- **Tips:**
  - Shows you in your element (editing, creating content)
  - Can be candid or professional
  - Should complement the profile image

### 3. Portfolio Project Images
- **Filenames:** `project1.jpg` through `project6.jpg`
- **Location:** `images/portfolio/`
- **Recommended Size:** 800x600px (landscape, 4:3 ratio)
- **Format:** JPG or PNG
- **Max File Size:** 200KB each
- **Content:** Screenshots or photos of your work

## üé® How to Create/Edit Images

### Option 1: Use Canva (Recommended)
1. Go to [canva.com](https://www.canva.com)
2. Create custom size:
   - Profile: 600x600px
   - About: 500x700px
   - Portfolio: 800x600px
3. Add your photo/screenshot
4. Apply filters or effects if needed
5. Download as JPG

### Option 2: Use Photo Editor
- **Windows:** Paint, Photos app
- **Mac:** Preview, Photos app
- **Mobile:** Snapseed, Lightroom Mobile
- **Online:** Photopea, Remove.bg

### Option 3: Use Screenshot Tools
For portfolio images of your work:
- **Windows:** Snipping Tool, Win + Shift + S
- **Mac:** Cmd + Shift + 4
- **Browser Extension:** Awesome Screenshot

## üí° Pro Tips

### For Best Results:

1. **Consistency**
   - Use similar lighting across all photos
   - Maintain consistent color grading
   - Match the mood/tone

2. **Quality**
   - Use high-resolution source images
   - Avoid pixelated or blurry images
   - Proper exposure and focus

3. **Branding**
   - Use your brand colors in designs
   - Add subtle watermarks if needed
   - Consistent visual style

4. **File Management**
   - Keep original high-res versions
   - Save optimized versions for web
   - Organize in folders by type

5. **Accessibility**
   - Good contrast for text overlays
   - Clear, easy to understand visuals
   - Professional and appropriate content

## ‚ùì Troubleshooting

### Image Not Showing?
- Check filename matches exactly (case-sensitive)
- Verify file is in correct folder
- Check file extension (.jpg vs .png)
- Open browser console for errors

### Image Too Large?
- Compress using TinyPNG
- Resize to recommended dimensions
- Convert to JPG if it's PNG

### Image Looks Pixelated?
- Use higher resolution source
- Don't enlarge small images
- Re-export at correct size

### Wrong Aspect Ratio?
- Crop to recommended dimensions
- Use Canva to resize properly
- Center important content before cropping
